For `italic` use `*` or `_`

```markdown
Words in this _sentence_ are in *italics*.
```

For `strong`/`bold` use `**` or `__`

```markdown
Words in this __sentence__ are in **bold**.
```

For `strikethrough` use `~~` (tilde)

```markdown
Strikethrough uses ~~two tildes~~.
```
