Images use the same syntax as [links](/s/links-in-markdown), but prepended with `!`

```markdown
![alt text](https://example.com/logo.png "Logo Title")
```

You can use reference style as well

```markdown
![alt text][logo]
.
.
.
[logo]: https://example.com/logo.png "Logo Title"
```
